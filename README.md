# D2R Loot Filter

Loot filters streamline Diablo II: Resurrected (Battle.net) by hiding useless items during mid- and early lategame, reducing clutter and focusing drops.

## Purpose
Hides low-value items like white/normal gear and junk bases in midgame through early lategame. Improves loot readability and highlights runes/jewels/elite bases.

## Installation

### Quick Method
1. Copy contents from [Filter.txt](Filter.txt).
2. In D2R, open **Esc > Loot Filter > Import**, paste contents, and apply.

### Advanced Method (Mod)
1. Download via code button → Download ZIP.
2. Extract `D2RFilter-main` to game folder `...\Diablo II Resurrected\mods\` (create `mods` folder if needed) and rename it to `Filter`.
3. In Battle.net: **Game Settings > Additional command line arguments** → `-mod Filter`. [Restart client.]
